One operating function is Device Management. The function of device management is to keep track of all devices. It does this by using a task known as the I/O controller. it does a number of things from deciding whic process gets the device for how much time and when to allocating device effciency. 
Another operating function is File management. File management is basically your organizer it keeps everything nice and neat in usually a directory that is easy to navigate. It keeps track of information, uses, location, and status. It also decides who gets what resources and allocates and de-allocates resources.
